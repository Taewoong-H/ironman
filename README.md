# CSS 연습용 사이트 만들기
 이 레포는 반응형 웹사이트 및 CSS 와 javascript에 대한 연습을 하기 위해 만든 사이트이다.

 * 반응형(responsive) media query 연습
 * css positioning, display 연습
 * animation 및 transition
 * attribute 속성 이해
 * image 태그 float로 위치 조정 연습
 * javascript forEach와 this 구문의 이해
